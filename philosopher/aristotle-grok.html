01
BEGINNER
入門レベル：アリストテレスって誰？
FACT
  • 紀元前384年マケドニア北部スタゲイラ生まれ、322年エウボエア島没（享年61-62歳）
  • 17歳頃プラトンのアカデメイアに入学、20年間学んだ後師の死後に離れた
  • マケドニア王フィリッポス2世の求めでアレクサンドロス大王の家庭教師（BC343-340年頃）
  • アテナイに「リュケイオン（Lyceum）」を設立。現存著作は約30編（論理・自然・形而上・倫理・政治・詩学など）
INTERPRETATION
「現実主義・経験主義の祖」と位置づけられ、目に見える世界を詳細に観察・分類する姿勢が近代科学の基礎を築いた。プラトンの「天上界」重視に対し、「この世界」を深く理解しようとした点が特徴。
02
INTERMEDIATE
中級レベル：主要思想と方法
FACT
  • 四原因説：質料因・形相因・運動因・目的因の4つで事物を説明（『物理学』）
  • 形質論（Hylomorphism）：すべての事物は「質料（hyle）」と「形相（morphe）」の結合。魂も身体の形相として捉える
  • 論理学：三段論法の体系化。10カテゴリー（実体・量・質など）を整備
  • 倫理学：徳は「中庸（golden mean）」にあり習慣によって養われる。最高善は「幸福（eudaimonia）」（『ニコマコス倫理学』）
  • 政治学：人間は「政治的動物（zoon politikon）」。政体を正しい型とその腐敗型に分類
INTERPRETATION
  • イデア論批判：イデアは事物から分離されたものではなく事物の内にある形相——内在的形而上学が生まれた
  • 中庸の倫理は相対主義ではなく、状況に応じた「賢慮（phronesis）」による実践知を重視する現実主義
  • 目的因の重視は近代科学で一時退けられたが、現代のシステム思考や目的論的生物学で再評価されている
  • 女性・奴隷に関する記述は当時の慣習を反映。全体として「善き生」の探求は普遍的価値を持つ
03
ADVANCED
上級レベル：歴史的位置と影響
FACT
  • ヘレニズム期にリュケイオン学派が継続。中世アラビア（アヴェロエスなど）・キリスト教神学（トマス・アクィナス）を通じてヨーロッパに再導入
  • ルネサンス以降、経験科学の基礎として活用されたが、ガリレオ・ニュートンらにより一部の自然学が批判された
  • 20世紀に「アリストテレス復興」——分析哲学・徳倫理学・現象学などで影響
INTERPRETATION
  • ベーコン・デカルトはスコラ化したアリストテレスを批判。一方ハイデッガーは実存的側面（存在の問い）を再発見
  • 現代的意義：生物学・認知科学での目的論的説明、AIの倫理・徳のプログラミング、コミュニタリアニズム、『詩学』のカタルシスなど
  • 「万学の祖」としての包括性は、今日の学際的研究のモデル
DEEP DIVE 四原因説と現代科学
THEME // 四原因説（『物理学』第2巻）を現代に接続する

例：彫像（青銅像）で4つの原因を見る

01 // 質料因
質料因
青銅——何の素材でできているか
02 // 形相因
形相因
像の設計・形——どんな構造をしているか
03 // 運動因
運動因
彫刻家の行為——誰が・何が作ったか
04 // 目的因 ★最重要
目的因
美の表現・神殿への奉納——何のためにあるか
DEEP INTERP
単なる因果説明ではなく、事物の本質を多角的に理解する枠組み。近代科学は運動因・質料因に集中し目的因を排除したが、進化生物学・工学・医学では暗黙的に用いられている。
MODERN
IMPLICATIONS
AI / 生命倫理
「目的（goal）」をどう扱うか——設計目的と自律的目標の区別
環境問題
人間中心の目的因を超えた生態系のtelosを考える契機に
量子力学・複雑系
「全体性」「創発」の議論で形相因的な視点が再び重要視される
